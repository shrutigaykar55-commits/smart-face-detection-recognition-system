# Smart Face Detection and Recognition System

A real-time Face Detection and Recognition desktop application built using **Python, OpenCV, LBPH Face Recognizer, and Tkinter GUI**.

This project detects faces using Haar Cascade and recognizes registered users in real time through a webcam interface.

---

## Features

- Real-time face detection
- Face recognition using LBPH algorithm
- Modern GUI interface
- Start / Stop / Exit buttons
- Multiple face support
- Unknown face detection
- Confidence score display
- Dataset creation for new users
- Real-time webcam feed

---

- NumPy
